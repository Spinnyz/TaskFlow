# TaskFlow

TaskFlow is a simple command-line task manager built in Python using Object-Oriented Programming (OOP) principles.  
The entire project runs in a single Python file for simplicity and easy execution.

This project focuses on practicing class design, object interaction, and clean logic without external dependencies.
